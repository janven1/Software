## 01绘图命令

| 快捷键   | 功能            | 说明           |
| ----- | ------------- | ------------ |
| `L`   | 直线（LINE）      | 绘制直线段        |
| `C`   | 圆（CIRCLE）     | 绘制圆          |
| `REC` | 矩形（RECTANGLE） | 绘制矩形         |
| `A`   | 圆弧（ARC）       | 绘制圆弧         |
| `PL`  | 多段线（PLINE）    | 绘制连续线段（可带宽度） |
| `POL` | 多边形（POLYGON）  | 绘制正多边形       |
| `EL`  | 椭圆（ELLIPSE）   | 绘制椭圆或椭圆弧     |
| `H`   | 填充（HATCH）     | 填充图案或颜色      |
## 02修改命令
| 快捷键   | 功能          | 说明         |
| ----- | ----------- | ---------- |
| `E`   | 删除（ERASE）   | 删除选中对象     |
| `CO`  | 复制（COPY）    | 复制对象       |
| `M`   | 移动（MOVE）    | 移动对象       |
| `RO`  | 旋转（ROTATE）  | 旋转对象       |
| `SC`  | 缩放（SCALE）   | 缩放对象       |
| `TR`  | 修剪（TRIM）    | 修剪多余线段     |
| `EX`  | 延伸（EXTEND）  | 延伸线段到边界    |
| `MI`  | 镜像（MIRROR）  | 镜像复制对象     |
| `O`   | 偏移（OFFSET）  | 创建平行线或同心结构 |
| `F`   | 倒圆角（FILLET） | 添加圆角       |
| `CHA` | 倒角（CHAMFER） | 添加斜角       |
| `X`   | 分解（EXPLODE） | 分解块或多段线    |
## 03标注命令
| 快捷键   | 功能                | 说明        |
| ----- | ----------------- | --------- |
| `DLI` | 线性标注（DIMLINEAR）   | 水平/垂直尺寸标注 |
| `DAL` | 对齐标注（DIMALIGNED）  | 斜线尺寸标注    |
| `DRA` | 半径标注（DIMRADIUS）   | 标注圆或圆弧半径  |
| `DDI` | 直径标注（DIMDIAMETER） | 标注圆或圆弧直径  |
| `DAN` | 角度标注（DIMANGULAR）  | 标注角度      |
| `LE`  | 引线标注（QLEADER）     | 添加注释引线    |
## 04视图控制
| 快捷键         | 功能                 | 说明             |
| ----------- | ------------------ | -------------- |
| `Z->空格->E`  | 全图缩放（ZOOM EXTENTS） | 显示全部图形         |
| `Z ->空格->W` | 窗口缩放（ZOOM WINDOW）  | 局部放大指定区域       |
| `P`         | 平移（PAN）            | 移动视图（按住鼠标中键也可） |
| `RE`        | 重生成（REGEN）         | 刷新视图（解决显示问题）   |
## 05图层与属性
| 快捷键  | 功能             | 说明             |
| ---- | -------------- | -------------- |
| `LA` | 图层管理（LAYER）    | 打开图层管理器        |
| `MA` | 格式刷（MATCHPROP） | 复制对象属性（如颜色、线型） |
| `PU` | 清理（PURGE）      | 删除未使用的图层、块等    |
## 06其他使用命令
| 快捷键      | 功能          | 说明         |
| -------- | ----------- | ---------- |
| `B`      | 创建块（BLOCK）  | 将对象组合成块    |
| `I`      | 插入块（INSERT） | 插入已保存的块或文件 |
| `CTRL+S` | 保存          | 快速保存当前文件   |
| `CTRL+Z` | 撤销          | 回退上一步操作    |
| `CTRL+Y` | 重做          | 恢复撤销的操作    |
