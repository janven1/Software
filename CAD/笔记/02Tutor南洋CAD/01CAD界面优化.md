## 04设置选项调整
| 序号  |                                内容                                | 介绍                                                                                                                                                                                   | 评价  |
| :-: | :--------------------------------------------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :-: |
| 01  |   [设置-文件](https://www.bilibili.com/video/BV1aT4y1B7oY?t=108.5)   | 后续讲解样板设置、自动保存                                                                                                                                                                        |     |
| 02  |   [设置-显示](https://www.bilibili.com/video/BV1aT4y1B7oY?t=119.7)   | `十字光标大小=>18`，其他无需调整<br>`颜色->颜色=>黑色`(我设置为白色且单独设置了`颜色主题=>浅色`)<br>`布局元素`不用修改，直到布局空间出图(后续有专门讲解)                                                                                          |     |
| 03  | [设置-打开和保存](https://www.bilibili.com/video/BV1aT4y1B7oY?t=171.3)  | `文件保存->另存为->**2004/2000`：能够让别人打开你的图纸<br>`文件安全措施->取消勾选 “自动保存”+“每次保存时均创建备仍副本”`<br>(后续有专门讲解-自动保存以及创建副本)<br>`ObjectARX应用程序->自定义对象的代理图像=>"不显示代理图形"->取消勾选"显示代理信息'对话框'"`：在我们打开别人图纸的时候，不总是弹窗 |     |
| 04  |                           设置-打印和发布/系统                            | 无需调整                                                                                                                                                                                 |     |
| 05  | [设置-用户系统配置](https://www.bilibili.com/video/BV1aT4y1B7oY?t=230.8) | `Windows标准操作`切忌不要取消勾选，对于操作很重要<br>`插入比例=>毫米`我国大多数图纸都为毫米为单位<br>其余和UP保持相同即可                                                                                                             |     |
| 06  |   [设置-绘图](https://www.bilibili.com/video/BV1aT4y1B7oY?t=253.0)   | 和UP保持相同即可：勾选相应的功能；<br>2026版本无需调整                                                                                                                                                     |     |
| 07  |                             设置-三维建模                              | 无需调整                                                                                                                                                                                 |     |
| 08  |  [设置-选择集](https://www.bilibili.com/video/BV1aT4y1B7oY?t=259.6)   | `拾取框大小`不要调太大，大概1/3不到的位置；<br>`选择集模式`与UP勾选相同(26版保持默认)，<br>一旦多勾选，可能无法多选对象<br>`选择集模式->允许按住并拖动套索`可以取消勾选<br>其他保持默认                                                                         |     |
| 09  |   [设置-配置](https://www.bilibili.com/video/BV1aT4y1B7oY?t=309.4)   | `添加到列表`可将当前配置进行保存(26破解版无法添加，报错缺少路径)<br>`重置`将整个CAD设置恢复到出厂状态                                                                                                                           |     |
