# 00国内可用docker镜像源

- `docker-desktop->setting->Docker Engine`修改配置文件中“"registry-mirrors"的网站名

```
{
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false,
  "registry-mirrors": [
    "https://docker.m.daocloud.io",
    "https://docker.1panel.live",
    "https://docker.1ms.run",
    "https://hub.rat.dev"
  ]
}
```
