### 02普通
| 序号  |        内容         | 介绍                     |
| :-: | :---------------: | ---------------------- |
| 01  | `:help <command>` | 查看命令的官方文档              |
| 02  |      `<C-q>`      | 能够进行列选中，其实原本应该是`<C-v>` |
| 03  |        `L`        | 光标移动到当前屏幕底部            |
| 04  |        `M`        | 光标移动到当前屏幕中间            |
| 05  |        `H`        | 光标移动到当前屏幕顶部            |
| 06  |    `t+<char>`     | 光标向后移动到当前行`<char>`前一位  |
| 07  |    `T+<char>`     | 光标向前移动到当前行`<char>`后一位  |
| 08  |        `r`        | 光标当前字母能够修改为其他字母        |
| 09  |        `.`        | 重复上一个文本编辑内容            |
|     |                   |                        |
