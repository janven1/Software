### 04PPT字体大小参考
参考Deepseek

| 序号  |   类型    | 字体大小/pt  |           示例           |
| :-: | :-----: | :------: | :--------------------: |
| 01  |   主标题   |  28-44   |      如封面页用36-44pt      |
| 02  |   节标题   |  24-32   |     如“研究背景”“数据分析”      |
| 03  |  普通文本   |  18-24   | 确保后排观众看清，**最小不低于18pt** |
| 04  |  重点强调   | 加粗或24-28 |        如关键结论、数据        |
| 05  |  图表标题   |  20-24   |                        |
| 06  | 数据标签/图例 |  16-20   |     **最小不低于14pt**      |
|     |         |          |                        |
