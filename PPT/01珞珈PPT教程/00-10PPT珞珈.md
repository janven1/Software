### 02职场有这么一句话
做Word的不如做Excel的；
做Excel的不如做PPT的；
做PPT的不如讲PPT的；
讲PPT的不如听PPT的；

| 序号  |   原文    |                              翻译                               |
| :-: | :-----: | :-----------------------------------------------------------: |
| 01  | 做Word的  |                        基层员工，比方说一些行政文员                         |
| 02  | 做Excel的 |                HR，一般来说就是就是财务会计<br>这些岗位呢属于中间岗位                 |
| 03  |  做PPT的  |         通常一些外派的一些角色<br>比方说：商务、运营、老板助理<br>做项目能让公司挣钱的东西         |
|     |  讲PPT的  |  通常都会在一些重要场合上<br>运用动人的演讲技巧<br>征服我们的观众或者说是客户<br>一般是项目的负责人，老板   |
|     |  听PPT的  | 老板要征服的对象<br>老板这个公司的好与坏呢<br>基本上都是听着他们的这个生杀大权<br>一般是客户、投资人或者董事会 |
